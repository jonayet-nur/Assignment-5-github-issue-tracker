## 1.What is the difference between var, let, and const?

### var is function-scoped, can be redeclared, and can be reassigned.

### let is block-scoped, cannot be redeclared, but it can be reassigned.

### const is also block-scoped, and it cannot be redeclared or reassigned after initialization.

## 2. What is the spread operator (...)?

### The spread operator  in JavaScript  is used to expand or spread elements of an array, object, or iterable into individual elements.

## 3. What is the difference between map(), filter(), and forEach()?

### map() creates a new array by applying a function to every element of the oiginal.
### filter() creates a new array containing only elements that pass a test.
### forEach() doesn't return anything. It only executes code for every item that is in an array.

## 4. What is an arrow function?
### An arrow function is a shorter, more modern way to write functions in JavaScript, introduced in ES6 . it as a shorthand that makes  code clean  and more readable.
